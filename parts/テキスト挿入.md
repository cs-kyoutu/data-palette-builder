# テキスト挿入テンプレート

## フォーマット

```
『テキスト挿入』
「{column}」の《{position}》に""{text}""を挿入
《{save_method}》にて保存
クレンジングタスクの保存名を""{task_name}""にする
```

## フォーマット（文頭+文末の両方）

```
『テキスト挿入』
「{column}」の《文頭》に""{text_head}""、《文末》に""{text_tail}""を挿入
《{save_method}》にて保存
クレンジングタスクの保存名を""{task_name}""にする
```

## 変数

| 変数 | 選択肢 |
|------|--------|
| position | 文頭 / 文末 |
| text | 挿入するテキスト |
| save_method | 上書き保存 / 名前を付けて保存 |
| task_name | 保存名 |
