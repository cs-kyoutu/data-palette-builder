# テンプレート 横持ち→縦持ち変換

## フォーマット

```
『テンプレート『顧客ごとに横持ちのデータを縦に並べて変換』』
[縦持ちにしたいカラム]に{horizontal_columns}を選択し、[適用]を押下する
[縦持ちにしたいカラム以外に残したいカラム]に{keep_columns}を選択し、[適用]を押下する
```

## 変数

| 変数 | 説明 | 例 |
|------|------|-----|
| horizontal_columns | 縦持ちにするカラム（「」区切り） | 「昨年1月」「昨年2月」...「今年12月」 |
| keep_columns | 残すカラム（「」区切り） | 「CUSTOMER_CODE」 |

## 注意
- 変換後、自動的に「縦持ちカラム」という名前のカラムが作成される（元の値が入る）
- 元のカラム名は消える
- 「縦持ちカラム」から必要な情報を抽出する（年月等）
