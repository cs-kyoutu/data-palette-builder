# 分割テンプレート

## フォーマット

```
『分割』
対象カラム: {column}
区切り文字: {delimiter}
{direction}
分割数: {count}
新カラム名: {new_columns}
```

## 変数

| 変数 | 選択肢 | 例 |
|------|--------|-----|
| column | 対象カラム | リレーション項目_10 |
| delimiter | 区切り文字 | , / - / スペース |
| direction | 左から / 右から | 左から |
| count | 分割数 | 10 |
| new_columns | 新カラム名（カンマ区切り） | 商品ID_1, 商品ID_2, ..., 商品ID_10 |
