# カラム名変更テンプレート

## フォーマット（単一）

```
『カラム名の変更』
「{old_name}」を""{new_name}""に変更する
```

## フォーマット（複数）

```
『カラム名の変更』
「{old_name_1}」を""{new_name_1}""に、「{old_name_2}」を""{new_name_2}""に変更する
```

## フォーマット（多数 - 改行区切り）

```
『カラム名の変更』
「{old_name_1}」を""{new_name_1}""に変更する
「{old_name_2}」を""{new_name_2}""に変更する
「{old_name_3}」を""{new_name_3}""に変更する
```
