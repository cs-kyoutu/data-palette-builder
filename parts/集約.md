# 集約テンプレート

## フォーマット

```
『集約』
{group_keys}を集約のキーとして、{aggregations}で集約
```

## 変数

| 変数 | 説明 | 例 |
|------|------|-----|
| group_keys | グループ化カラム（「」区切り） | 「商品ID【商品】」 |
| aggregations | 集計設定（カラム名+集計方法） | 「売上金額【受注】」を《合計》 |

## 集計方法
カウント / ユニークカウント / 合計 / 平均 / 最大値 / 最小値 / 最新日時 / 最古日時 / 行結合

## 複数集計の場合

```
『集約』
{group_keys}を集約のキーとして、「{col_1}」を《{agg_1}》、「{col_2}」を《{agg_2}》で集約
```
