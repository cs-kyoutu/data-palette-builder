# 型変換テンプレート

## フォーマット（名前を付けて保存）

```
『型変換』
「{column}」を《{target_type}》に変換
型変換エラーの処理は、《{error_handling}》を選択
《名前を付けて保存》にて保存
クレンジングタスクの保存名を""{task_name}""にする
```

## フォーマット（上書き保存）

```
『型変換』
「{column}」を《{target_type}》に変換
型変換エラーの処理は、《{error_handling}》を選択
《上書き保存》にて保存
クレンジングタスクの保存名を""{task_name}""にする
```

## 変数

| 変数 | 選択肢 |
|------|--------|
| target_type | テキスト型 / 整数型 / 小数型 / 日付型 / 日時型 / 真偽値型 |
| error_handling | Nullに変換する / エラーにする |
| task_name | 保存名（通常はカラム名と同じ） |

## 注意
- 型変換は**1カラムにつき1回の保存が必要**
- 複数カラムの型変換は個別に実行する
- 上書き保存は過去1回以上クレンジング済みファイルのみ
