# 縦統合テンプレート

## フォーマット

```
『縦統合』
【{file_1}】と【{file_2}】{file_3_text}{file_4_text}を縦統合
重複データの設定は
[{duplicate_setting}]を選択し、[適用]を押下する
データファイルの保存名を""{save_as}""にする
統合データの更新設定は《{update_setting}》を選択し、[適用]を押下する
```

## 変数

| 変数 | 説明 |
|------|------|
| file_1〜4 | 統合するファイル名（最大4つ） |
| file_3_text | 3つ目がある場合「と【{file_3}】」、なければ空 |
| file_4_text | 4つ目がある場合「と【{file_4}】」、なければ空 |
| duplicate_setting | 重複データを除外する / 重複データを除外しない |
| save_as | 保存ファイル名 |
| update_setting | 更新する / 更新しない |

## 注意: 4ファイル制限
5ファイル以上の場合は分割して縦統合し、結果同士をさらに縦統合する。
