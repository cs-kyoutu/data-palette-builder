# 置換テンプレート

## フォーマット

```
『置換』
「{column}」の《{from_value}》を""{to_value}""に置換
《{save_method}》にて保存
クレンジングタスクの保存名を""{task_name}""にする
```

## 変数

| 変数 | 説明 | 例 |
|------|------|-----|
| column | 対象カラム | 合計売上金額 |
| from_value | 置換前の値 | NULL |
| to_value | 置換後の値 | 0 |
| save_method | 上書き保存 / 名前を付けて保存 |
| task_name | タスク保存名 | 合計売上金額 |
