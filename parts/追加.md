# 追加テンプレート

## フォーマット

```
『追加』
「{column}」の《{position}》を選択
《{data_type}》の""{default_value}""を追加
```

## フォーマット（加工処理実行日）

```
『追加』
カラム名: {column}
データ型: {data_type}
加工処理実行日カラム: チェック
```

## 変数

| 変数 | 選択肢 |
|------|--------|
| position | 右に追加 / 左に追加 |
| data_type | テキスト型 / 整数型 / 小数型 / 日付型 / 日時型 |
| default_value | 初期値（空文字の場合は""） |
