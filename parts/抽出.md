# 抽出テンプレート

## フォーマット

```
『抽出』
対象カラム: {column}
抽出方法: {method}
文字数: {length}
新カラム名: {new_column}
```

## 変数

| 変数 | 選択肢 | 例 |
|------|--------|-----|
| method | 先頭から抽出 / 中間を抽出 / 末尾から抽出 | 末尾から抽出 |
| length | 文字数（中間の場合はX〜Y） | 5 |
| new_column | 新カラム名 | 誕生月日 |

## 中間の場合

```
『抽出』
対象カラム: {column}
抽出方法: 中間を抽出
開始位置: {start}文字目
終了位置: {end}文字目
新カラム名: {new_column}
```
