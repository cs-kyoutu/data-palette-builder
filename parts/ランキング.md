# ランキングテンプレート

## フォーマット（グループ内）

```
『ランキング』
「{group_key}」をグループ化する単位として、「{sort_column}」を《{sort_order}》に《{tie_handling}》で順位付けする
```

## フォーマット（全体）

```
『ランキング』
「{sort_column}」を《{sort_order}》に《{tie_handling}》で順位付けする
```

## 変数

| 変数 | 選択肢 |
|------|--------|
| group_key | グループ化カラム名 |
| sort_column | ソート対象カラム名 |
| sort_order | 大きい順 / 小さい順 |
| tie_handling | 同率あり(順位飛ばしあり) / 同率なし |
