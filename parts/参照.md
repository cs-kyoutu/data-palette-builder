# 参照テンプレート

## フォーマット

```
『参照』
《{pattern}》を選択し、[適用]を押下する
「{group_key}」でグループ化、「{sort_column}」の《{sort_order}》でソートを行い
「{value_column}」を参照する
空文字/Nullの値を《{null_handling}》を選択し[適用]を押下する
クレンジングタスクの保存名を""{task_name}""にする
```

## 変数

| 変数 | 選択肢 |
|------|--------|
| pattern | 最初の値 / 最後の値 / 特定順番の値 / 前の行の値 / 後ろの行の値 / 累計の値 |
| group_key | グループ化カラム |
| sort_column | ソートカラム |
| sort_order | 昇順 / 降順 |
| value_column | 参照するカラム |
| null_handling | 無視する / 無視しない |
| task_name | 保存名 |
