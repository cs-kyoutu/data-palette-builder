# 四則演算テンプレート

## フォーマット（カラム×カラム）

```
『四則演算』
「{column_left}」《{operator}》「{column_right}」、[端数処理]は小数第《{decimal_place}》位を、《{rounding}》する
クレンジングタスクの保存名を""{task_name}""にする
表示方法《{keep_original}》を選択
```

## フォーマット（カラム×固定値）

```
『四則演算』
「{column_left}」《{operator}》""{fixed_value}""、[端数処理]は小数第《{decimal_place}》位を、《{rounding}》する
クレンジングタスクの保存名を""{task_name}""にする
表示方法《{keep_original}》を選択
```

## 変数

| 変数 | 選択肢 | 例 |
|------|--------|-----|
| operator | + / - / × / ÷ | × |
| decimal_place | 1 / 2 / 3 / 4 | 1 |
| rounding | 四捨五入 / 切り捨て / 切り上げ | 四捨五入 |
| keep_original | 残す / 残さない | 残す |
| fixed_value | 固定値（数値） | 100 / 1000 |

## 注意
- NULL含有時の結果はNULL（テキスト型）
- 0除算の場合もNULL
