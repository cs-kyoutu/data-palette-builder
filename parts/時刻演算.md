# 時刻演算テンプレート

## フォーマット

```
『時刻演算』
左辺カラム: {left_column}
演算子: {operator}
右辺カラム: {right_column}
単位: {unit}
新カラム名: {new_column}
```

## フォーマット（カスタム値）

```
『時刻演算』
「{column}」《{operator}》《カスタム：""{custom_value}""》　[計算結果]を《{unit}》で算出
クレンジングタスクの保存名を""{task_name}""にする
表示方法《{keep_original}》を選択
```

## 変数

| 変数 | 選択肢 | 例 |
|------|--------|-----|
| left_column | カラム名 or 本日の日付 | 今年の誕生日 / 現在日時 |
| operator | - / + | - |
| right_column | カラム名 or 本日の日付 or カスタム | 本日日付 / 最終カート投入日 |
| custom_value | 固定数値（カスタムの場合） | 1 / 3 / 5 |
| unit | 日 / 時間 / ヶ月 | ヶ月 |
| new_column | 新カラム名 | 1か月前_注文年月日 |
| keep_original | 残す / 残さない | 残す |
