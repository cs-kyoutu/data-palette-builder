# 時刻演算テンプレート

## フォーマット

```
『時刻演算』
左辺カラム: {left_column}
演算子: {operator}
右辺カラム: {right_column}
単位: {unit}
新カラム名: {new_column}
```

## 変数

| 変数 | 選択肢 | 例 |
|------|--------|-----|
| left_column | カラム名 or 本日の日付 | 今年の誕生日 / 現在日時 |
| operator | - / + | - |
| right_column | カラム名 or 本日の日付 or カスタム | 本日日付 / 最終カート投入日 |
| unit | 日 / 時間 | 日 |
| new_column | 新カラム名 | 誕生日までの日数 |
