# テンプレート「3つ以上のカラムやテキストを連結した値を追加」

## フォーマット

```
『テンプレート、3つ以上のカラムやテキストを連結した値を追加』
{columns}を選択する
[テキスト挿入]を押下し、カラムとカラムの間に""{separator}""を挿入し、[適用]を押下する
クレンジングタスクの保存名を""{task_name}""にする
表示方法《{keep_original}》を選択
```

## 変数

| 変数 | 説明 | 例 |
|------|------|-----|
| columns | 連結するカラム（「」区切り） | 「受注ID」「mall_name」「商品ID」 |
| separator | 区切り文字 | , / - / (なし) |
| task_name | 保存名 | 受注明細ID |
| keep_original | 残す / 残さない | 残す |

## 注意
- 通常の連結は2カラムまで。3カラム以上はこのテンプレートを使用
