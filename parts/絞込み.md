# 絞込みテンプレート

## フォーマット（単一条件）

```
『絞込み』
「{column}」が{condition}に絞り込む
```

## フォーマット（複数条件AND）

```
『絞込み』
「{column_1}」が{condition_1}
{logic}
「{column_2}」が{condition_2}
に絞り込む
```

## 条件パターン

| 条件タイプ | 書き方 | 例 |
|-----------|--------|-----|
| 完全一致 | ""{value}""《次と等しい》 | ""1""《次と等しい》 |
| 不一致 | ""{value}""に完全一致しない | ""1""に完全一致しない |
| 空文字ではない | 《空文字ではない》 | 《空文字ではない》 |
| 空文字 | 《空文字》 | 《空文字》 |
| 範囲 | ""{from}""~""{to}""《次の値の間》 | ""1""~""8""《次の値の間》 |
| 以上 | ""{value}""《次の値以上》 | ""9900000""《次の値以上》 |
| 以下 | ""{value}""《次の値以下》 | ""5""《次の値以下》 |
| 相対期間 | 《相対期間》[開始日]《{n}》《{unit}》《前》と[終了日]《{m}》《{unit}》《前》《次の期間にある》 | 《相対期間》[開始日]《30》《日》《前》と[終了日]《0》《日》《前》《次の期間にある》 |
