# 連結テンプレート

## フォーマット

```
『連結』
連結カラム1: {column_1}
連結カラム2: {column_2}
区切り文字: {separator}
新カラム名: {new_column}
元カラム: {keep_original}
```

## 変数

| 変数 | 説明 | 例 |
|------|------|-----|
| column_1 | 1つ目のカラム | 姓 |
| column_2 | 2つ目のカラム | 名 |
| separator | 区切り文字 | (スペース) / (なし) / - / / |
| new_column | 新カラム名 | 名前 |
| keep_original | 元カラムの扱い | 残さない / 残す |
