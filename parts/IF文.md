# IF文テンプレート

## フォーマット（値比較）

```
『IF文』
「{column}」が""{value}""《{condition}》の場合、""{true_value}""に変換
いずれの条件分岐にも該当しない場合、{else_value}に変換
```

## フォーマット（カラム値比較）

```
『IF文』
「{column_a}」が「{column_b}」《次のカラムの値に完全一致》の場合、""{true_value}""に変換
いずれの条件分岐にも該当しない場合、《空白》に変換
```

## フォーマット（複数条件）

```
『IF文』
「{column}」が""{value_1}""《{condition}》の場合、""{result_1}""に変換
「{column}」が""{value_2}""《{condition}》の場合、""{result_2}""に変換
いずれの条件分岐にも該当しない場合、""{else_value}""に変換
```

## フォーマット（期間判定 - 絶対期間+カラム参照）

```
『IF文』
《絶対期間》を選択
「{date_column}」が[開始日]「{start_column}」と[終了日]「{end_column}」《指定の期間に一致する》の場合、""{true_value}""に変換
いずれの条件分岐にも該当しない場合、""{false_value}""に変換
[開始日時/終了日時を含めない]にチェックを入れない
[期間指定にカラムを使用する]にチェックを入れる
```

## フォーマット（期間判定 - 開始のみ）

```
『IF文』
《絶対期間》を選択
「{date_column}」が[開始日]「{start_column}」《指定の期間に一致する》の場合、""{true_value}""に変換
いずれの条件分岐にも該当しない場合、""{false_value}""に変換
[開始日時/終了日時を含めない]にチェックを入れない
[期間指定にカラムを使用する]にチェックを入れる
```

## フォーマット（カンマ区切り複数値一致）

```
『IF文』
「{column}」が""{value_1},{value_2}""《次に完全一致(カンマ区切りで複数指定)》の場合、""{result}""に変換
```

## else_value
- 《空白》
- ""{具体的な値}""
- 《元の値のまま》
- 「{カラム名}」に変換（他カラムの値を参照）
