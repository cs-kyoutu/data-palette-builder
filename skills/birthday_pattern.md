# 誕生日施策パターン（10ステップ、省略厳禁）

誕生日N日前のセグメント条件を算出するための固定パターン。全10ステップを必ず実施する。

---

## 処理フロー

### [A] テンプレート 生年月日→年齢算出
- テンプレート → 生年月日から年齢を算出
- 生年月日カラムを選択 → 新カラム「年齢」

### [B] 型変換: 生年月日 → テキスト型
- 生年月日カラムをテキスト型に変換

### [C] 抽出（末尾5文字）→ 誕生月日
- 末尾から5文字を抽出
- 例: "1990-03-15" → "03-15"
- 新カラム名: 誕生月日

### [D] IF文: うるう年対応
- 条件: 誕生月日 = "02-29"
- 一致時: "02-28"
- 不一致時: そのまま
- 2/29生まれの人を2/28として処理する

### [E] 追加: 本日日付
- カラム追加（日付型）
- 「加工処理実行日カラム」チェックボックスをON
- 新カラム名: 本日日付

### [F] 型変換: 本日日付 → テキスト型
- 本日日付カラムをテキスト型に変換

### [G] 抽出（先頭5文字）→ 本年
- 先頭から5文字を抽出
- 例: "2026-03-23" → "2026-"
- 新カラム名: 本年

### [H] 連結: 本年 + 誕生月日 → 今年の誕生日テキスト
- 連結カラム1: 本年
- 連結カラム2: 誕生月日（うるう年対応済み）
- 新カラム名: 今年の誕生日テキスト

### [I] 型変換: テキスト → 日付型 → 今年の誕生日
- 今年の誕生日テキストを日付型に変換
- 新カラム名: 今年の誕生日

### [J] 時刻演算: 今年の誕生日 - 本日日付 → 誕生日までの日数
- 左辺: 今年の誕生日
- 演算子: -
- 右辺: 本日日付
- 単位: 日
- 新カラム名: 誕生日までの日数

---

## 使い方

この後、絞込みで「誕生日までの日数 = N」や「誕生日までの日数 が N〜M の間」等で対象セグメントを作成する。
