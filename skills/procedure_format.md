# 手順書出力フォーマット仕様

Phase2（手順書生成）で必ず参照する。このフォーマットから逸脱してはいけない。

---

## 全体構成

手順書は**データファイル単位**でセクション分けする。

```
【データファイル名A】
（このファイルの作成手順）

【データファイル名B】
（このファイルの作成手順）
```

---

## 加工セクションの書き方

### ヘッダー
どのファイルを加工するか明記：
```
【作成するファイル名】
　　【加工元ファイル名】を加工する
```

### 加工ステップは連番
```
1. 『操作名』
　　操作内容の詳細
2. 『操作名』
　　操作内容の詳細
```

### 最後に保存
```
　　データファイル名を""保存名""にして保存する。
```

---

## 統合セクションの書き方

```
【作成するファイル名】
　　『横統合』
　　【左ファイル】と【右ファイル】を「左キー」と「右キー」を統合キーとして、《統合方法》
　　残すカラムは、
　　【左ファイル】：「カラム1」、「カラム2」、「カラム3」
　　【右ファイル】：「カラム4」、「カラム5」
　　データファイルの保存名を""保存名""にする
　　レコードが重複した際の処理方法は
　　《統合処理をエラーにする》を選択し、[適用]を押下する
　　統合データの更新設定は《更新する/更新しない》を選択し、[適用]を押下する
```

統合方法は必ずb→dash名称：
- 《先に選択したデータに対して統合する》
- 《共通のデータのみを統合する》
- 《全てのデータを統合する》
- 《後に選択したデータに対して統合する》

---

## 各操作の記載フォーマット

### 絞込み
```
　　『絞込み』
　　「カラム名」が""値""《条件》に絞り込む
```
複数条件：
```
　　『絞込み』
　　「カラム1」が""値1""《条件1》
　　AND
　　「カラム2」が《空文字ではない》
　　に絞り込む
```

### IF文
```
　　『IF文』
　　「カラム」が""値""《条件》の場合、""結果""に変換
　　いずれの条件分岐にも該当しない場合、《空白》に変換
```
カラム名変更が続く場合：
```
　　→作成したカラム名を「新カラム名」に変更する
```

### IF文（期間判定）
```
　　『IF文』
　　《絶対期間》を選択
　　「日付カラム」が[開始日]「開始カラム」と[終了日]「終了カラム」《指定の期間に一致する》の場合、""結果""に変換
　　いずれの条件分岐にも該当しない場合、""結果""に変換
　　[開始日時/終了日時を含めない]にチェックを入れない
　　[期間指定にカラムを使用する]にチェックを入れる
```

### 参照
```
　　『参照』
　　《パターン》を選択し、[適用]を押下する
　　「グループ化カラム」でグループ化、「ソートカラム」の《ソート順》でソートを行い
　　「参照カラム」を参照する
　　空文字/Nullの値を《無視する》を選択し[適用]を押下する
　　クレンジングタスクの保存名を""保存名""にする
```

### 名寄せ
```
　　『名寄せ』
　　「キー1」「キー2」を名寄せするキーとして、「優先カラム」を《優先順》に名寄せする
```

### 集約
```
　　『集約』
　　「キー1」「キー2」を集約のキーとして、「カラムA」を《集計方法A》、「カラムB」を《集計方法B》で集約
```

### ランキング
```
　　『ランキング』
　　「グループ化カラム」をグループ化する単位として、「ソートカラム」を《ソート順》に《同率設定》で順位付けする
```

### 型変換
```
　　『型変換』
　　「カラム」を《変換先型》に変換
　　型変換エラーの処理は、《エラー処理》を選択
　　《保存方法》にて保存
　　クレンジングタスクの保存名を""保存名""にする
```

### 連結
```
　　『連結』
　　「カラム1」と「カラム2」を選択
　　[テキスト挿入]を押下し、カラムとカラムの間に""区切り文字""を挿入し、[適用]を押下する
　　クレンジングタスクの保存名を""保存名""にする
　　表示方法《残す/残さない》を選択
```

### テキスト挿入
```
　　『テキスト挿入』
　　「カラム」の《文頭/文末》に""テキスト""を挿入
　　《保存方法》にて保存
　　クレンジングタスクの保存名を""保存名""にする
```

### 抽出
```
　　『抽出』
　　「カラム」の《先頭/中間/末尾》から""N""文字目までを《データ型》で抽出
　　《保存方法》にて保存
　　クレンジングタスクの保存名を""保存名""にする
```

### 時刻演算
```
　　『時刻演算』
　　「左辺カラム」《演算子》「右辺カラム」、[計算結果]を《単位》で算出
　　クレンジングタスクの保存名を""保存名""にする
　　表示方法《残す》を選択
```
カスタム値の場合：
```
　　『時刻演算』
　　「カラム」《-》《カスタム：""値""》　[計算結果]を《単位》で算出
```

### 四則演算
```
　　『四則演算』
　　「左辺カラム」《演算子》「右辺カラム」、[端数処理]は小数第《N》位を、《四捨五入/切り捨て/切り上げ》する
　　クレンジングタスクの保存名を""保存名""にする
　　表示方法《残す/残さない》を選択
```

### 置換
```
　　『置換』
　　「カラム」の《置換前値》を""置換後値""に置換
　　《保存方法》にて保存
　　クレンジングタスクの保存名を""保存名""にする
```

### 追加
```
　　『追加』
　　「基準カラム」の《右に追加》を選択
　　《データ型》の""初期値""を追加
```
加工処理実行日の場合：
```
　　『追加』
　　「基準カラム」の《右に追加》を選択
　　《日付型/日時型》を選択し、[加工処理実行日のカラムを追加する]にチェックを入れる
```

### 削除
```
　　『削除』
　　「カラム1」、「カラム2」、「カラム3」を削除する
```

### カラム名の変更
```
　　『カラム名の変更』
　　「旧名」を""新名""に変更する
```

### 分割
```
　　『分割』
　　「カラム」を[区切り文字]""文字""で、[開始位置]《左/右》から《N》列に分割する
　　クレンジングタスクの保存名を"名前1"、"名前2"にする
```

### 複製
```
　　『複製』
　　「カラム」を複製
　　→カラム名を「新名」に変更
```

### 書式変換
```
　　『書式変換』
　　「カラム」を選択、[オプション(歯車アイコン)]を押下し、《変換方式》に変換
　　《保存方法》にて保存
　　クレンジングタスクの保存名を""保存名""にする
```

### 除外
```
　　『除外』
　　「カラム」の《全ての》""文字列""を除外
　　《保存方法》にて保存
　　クレンジングタスクの保存名を""保存名""にする
```

### テンプレート：縦持ちを横持ちに変換
```
　　『テンプレート『顧客ごとに縦持ちのデータを横に並べて変換』』
　　「キー1」「キー2」を集約のキーとして選択し、[適用]を押下する
　　[横並びにしたいカラム]に「カラムA」、「カラムB」の順で選択し、[適用]を押下する
　　[並び順カラム]に「ソートカラム」《ソート順》を選択し、[適用]を押下する
　　[横並びにしたいカラムの数]を、カラムの並び順で設定した順番をもとに、上位《N》位まで横に並べるとし、[適用]を押下する
```

### テンプレート：金額カンマ区切り
```
　　テンプレート『金額をカンマ区切りにした値へ変換』
　　「カラム」が金額に該当するカラムに選択され、「カラム」に上書き保存されました
```

### テンプレート：曜日算出
```
　　テンプレート『日付型カラムから「曜日」を算出』
　　「日付カラム」が日付型カラムに選択され、「曜日カラム名」が追加されました
```

### テンプレート：3カラム以上連結
```
　　『テンプレート、3つ以上のカラムやテキストを連結した値を追加』
　　「カラム1」「カラム2」「カラム3」を選択する
　　[テキスト挿入]を押下し、カラムとカラムの間に""区切り文字""を挿入し、[適用]を押下する
　　クレンジングタスクの保存名を""保存名""にする
　　表示方法《残す/残さない》を選択
```

---

## インデントルール
- セクション見出し【】は字下げなし
- 加工元ファイルの記載は全角スペース2つ
- 操作名と詳細は全角スペース2つ
- 統合の残すカラム等は全角スペース2つ
